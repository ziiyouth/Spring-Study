### Repository-Rule

> 1) [Commit Rule Example]
- **Basic Commit rule** : <br>
   -  `[이름] : Ch1 실습` 
   - (ex) `[지영] : Ch3 실습` 
   
 <br>
 
- **Pull Request rule** : <br>
    <br>
   - `Pull Request` 제목 : `[이름] : 내용이나 수정사항`
      - (ex) `[지영] : Ch1 실습 코드` <br><br>
   - `Pull Request` 내용 : 
      - 자유롭게! 내가 공부한 내용 혹은 느낀 점, 트러블슈팅도 좋아요!
